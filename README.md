## Instalar ffmpeg para converção do áudio

### Linux

```bash
      sudo apt-get install ffmpeg
```

### Windows

[Site FFmpeg](https://ffmpeg.org/download.html#build-windows)

## Instalar dependências

```bash
      npm install
```

### Criar pastas _input_ e _output_ na raíz do projeto

```bash
      mkdir input output
```

```bash
      npm run dev
```
